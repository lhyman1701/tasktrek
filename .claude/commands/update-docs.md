# Update Documentation Command

Review and update project documentation after completing work:

1. **CLAUDE.md Updates**
   - Add new patterns discovered
   - Update architecture if changed
   - Add to "Recent Decisions" if applicable

2. **Component CONTEXT.md**
   - Update component documentation if modified
   - Add new conventions or patterns
   - Document new dependencies

3. **Reference Files**
   - Mark completed tasks in ROADMAP.md
   - Close bugs in BUGS.md if fixed
   - Mark refactors complete in REFACTORS.md

4. **Task Files**
   - Mark all checkboxes complete
   - Add "Discoveries During Implementation"
   - Update status to "Done"

Show what documentation updates you're making before executing.
