# Next Task Command

## Workflow

1. **Read ROADMAP**
   - Open and read `reference/ROADMAP.md`
   - Identify the next uncompleted high-priority task

2. **Load Task Specification**
   - Find the corresponding task file in `tasks/`
   - Read the complete task specification
   - Confirm you understand all requirements

3. **Verify Prerequisites**
   - Check that all dependencies are complete
   - Verify you have necessary context
   - Confirm no blockers exist

4. **Begin Implementation**
   - Follow the task file step-by-step
   - Mark progress checkboxes as you complete steps
   - Stop after each step for review

5. **Update Progress**
   - Update the task file with current status
   - Note any discoveries or decisions
   - Document blockers if encountered

Remember: Work on ONE task at a time. Use `/clear` after completing each task to maintain focus.
